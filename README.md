#git  

##config 的三个作用域  

###缺省等于local  
###$ git config --local 只对某个仓库有效  
###$ git config --global 当前所有仓库有效 
###$ git config --system 对系统所有登陆的用户有效

###显示config 的配置， 加 --list
###$ git config --list --local
###$ git config --list --global
###$ git config --list --system
